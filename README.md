🧮 Simple Arithmetic Calculator
A simple and responsive calculator web application built using HTML, CSS, and JavaScript.
This project performs basic arithmetic operations with a clean modern UI and keyboard support.

🚀 Features
✔ Addition
✔ Subtraction
✔ Multiplication
✔ Division
✔ Keyboard Support
✔ Clear Display Function
✔ Delete Last Character
✔ Decimal Number Support
✔ Responsive Design
✔ Input Validation

🛠 Technologies Used
HTML5
CSS3
JavaScript
